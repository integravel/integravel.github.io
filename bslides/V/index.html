Geração V

Quase 7 anos se passaram da geração IV, quando com o advento da IA para programar, decidi avançar nesse projeto com uma versão melhor e que rodasse como web app com layout ajustado pra smartphone. Dessa vez, já bem amadurecida a ideia, desisti dos níveis possivelmente sem solução. Em vez disso, criei um sistema de pontuação. 1 estrela se o desafio for resolvido com mais jogadas do que o mínimo necessário. 2 estrelas se o desafio for resolvido com a quantidade de jogadas igual ao mínimo necessário. 3 estrelas se o desafio for resolvido com menos jogadas que o mínimo necessário. Você pode pensar, como é possível resolver com menos jogadas do que o mínimo? Daí entra o Pensamento Computacional, o jogador pode usar uma vez cada uma dessas ações em casa desafio: criar um bloco numa posição específica, apagar um bloco numa posição específica. Assim, a ideia é não só encontrar uma solução ótima, mas determinar como podemos com o mínimo de alterações na grid, achar uma solução ainda melhor.
