Demonstração — Mudança de Base do Logaritmo

Teorema. Para \(a,b > 0\), \(a \neq 1\), \(b \neq 1\), vale:

\[ \log_a x = \frac{\log_b x}{\log_b a} \]

Demonstração.

Seja \(y = \log_a x\).
Pela definição de logaritmo, temos \(a^y = x\).
Como \(y = \log_a x\), concluímos que:
\[ \log_a x = \frac{\log_b x}{\log_b a} \]
